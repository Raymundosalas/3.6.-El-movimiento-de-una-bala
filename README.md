# Proyecto: Disparo de Bala en Unity 2D

## Descripción
Este proyecto consiste en la implementación de un sistema de disparo en un videojuego 2D desarrollado en Unity. Se simula la trayectoria de una bala mediante el uso de físicas en 2D.

## Características
- Disparo de proyectiles en dirección del jugador
- Movimiento basado en físicas (Rigidbody2D)
- Destrucción automática de balas
- Detección de colisiones

## Tecnologías utilizadas
- Unity 2D
- Lenguaje C#
- Motor de físicas de Unity

## Funcionamiento
El jugador puede disparar proyectiles presionando el botón de disparo. Las balas se instancian en un punto específico (FirePoint) y se desplazan en línea recta aplicando una fuerza.

## Estructura del proyecto
- Bullet.cs: Controla el comportamiento de la bala
- PlayerShooting.cs: Controla el disparo del jugador

## Instrucciones de uso
1. Abrir el proyecto en Unity
2. Ejecutar la escena principal
3. Presionar clic izquierdo para disparar

## Conclusión
En este proyecto aprendí a implementar mecánicas básicas de disparo en Unity 2D utilizando físicas. También comprendí cómo instanciar objetos dinámicamente y controlar su comportamiento mediante scripts en C#.
